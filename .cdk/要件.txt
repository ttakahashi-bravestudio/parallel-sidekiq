# 要件
このプロジェクトのコードを、AWSのECSにデプロイするCDKを書いてください。

# デプロイするもの
- VPC
- ECS(auto scaling)
- ECR
- RDS(MySQL)
- CloudWatch
- CodePipeline
- CodeBuild
- CodeDeploy
- Application Load Balancer
- WAF
- KMS(暗号化キー)
- NAT Gateway

# 構築について
ソースコードは"infra"ディレクトリに作成する
Typesriptで書く
パラメータとしてGithubのコネクションを指定できるようにしてください
パラメーターで指定したS3にアクセスできるロールをECSに追加してください
ECSのアーキテクチャはARM64で構築してください
パイプラインはブルーグリーンデプロイで構築してください
パイプラインはDockerfile.production をビルドしてECRにプッシュするようにしてください（プレフィックスも付けて）
ステージングとプロダクションの2つの環境を作成してください（パラメータで指定できるようにしてください）
VPCは、パブリック、プライベート、アイソレートを作り、ECSはプライベート、RDSはアイソレートに配置してください
ネットワークにあるものはできるだけKMSで暗号化してください
コメントは多めに書いてください
